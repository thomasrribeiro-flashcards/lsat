# lsat

Flashcards for the LSAT (Law School Admission Test) — for spaced repetition review via [flashcards](https://github.com/thomasrribeiro/flashcards).

Reflects the modern (2026) LSAT format: two scored Logical Reasoning sections + one scored Reading Comprehension section + one unscored experimental section, administered online via LSAC LawHub. Logic Games (Analytical Reasoning) was removed from the scored test in August 2024.

## Coverage

| File | Topic |
|------|-------|
| 01_overview | Test structure, scoring, sections, timing, writing sample, online format |
| 02_argument_structure | Premises, conclusions, indicators, sub-conclusions, counterpremises |
| 03_conditional_logic | Sufficient/necessary, contrapositive, mistaken reversal/negation, chains |
| 04_quantifiers | Some/most/all, valid combinations, negating quantifiers |
| 05_causal_reasoning | Correlation vs causation, alternative causes, strengthen/weaken causal arguments |
| 06_assumption_questions | Necessary vs sufficient assumption, negation test, supporter/defender |
| 07_strengthen_weaken | Strengthen, weaken, evaluate the argument, common traps |
| 08_inference_questions | Must be true, most strongly supported, cannot be true, fact-test |
| 09_method_role_principle | Method of reasoning, role in argument, principles, parallel reasoning |
| 10_flaws | Twelve canonical LSAT flaws: causal, conditional, equivocation, sample, etc. |
| 11_reading_comprehension | Passage types, question types, comparative reading, RC strategy |
| 12_study_plan | Prep schedule, drilling, blind review, plateaus, test-day strategy |

## Goal

Designed to take a serious test-taker to an exceptional score (170+). Cards emphasize:
- Pre-phrasing answers before reading choices
- Recognizing canonical wrong-answer trap patterns
- Pacing tactics (35-minute sections, 80 sec/question)
- Blind Review and the wrong-answer journal as the highest-leverage prep habits
